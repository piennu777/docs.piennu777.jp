# Session Token

### セッショントークンの発行

セッションを発行することができるのは、アカウントへログインするときに発行されます。

### セッショントークンのリセット

セッショントークンをリセットするには、[設定](https://tn.piennu777.jp/settings/)へアクセスし、「セッショントークンをリセットする」を押すことでセッショントークンをリセットすることができます。
